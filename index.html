2019-10-18

关于node、git和markdown的使用总结

1.node

Node.js 是一个基于 Chrome V8 引擎的javascript运行环境。 Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型。

NPM（node package manager），通常称为node包管理器。顾名思义，它的主要功能就是管理node包，包括：安装、卸载、更新、查看、搜索、发布等。

  • 初始化

    npm init

  • 安装 npm 上发布的某个版本、某个tag、某个版本区间的模块，甚至安装本地目录、压缩包和 git/github 的库作为依赖。

    npm install(简写:npm i)

  • hexo

    npm install hexo -g 安装

    npm update hexo -g 升级

    hexo n "..." 新建文章

    hexo p(publish)

    hexo g (generate) 生成静态文件

    hexo s (server) 启动服务预览

    hexo d (deploy) 部署

2.git

是一个开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本管理。

  • 在目录中创建新的 Git 仓库

    git init

  • 拷贝一个 Git 仓库到本地，让自己能够查看该项目或进行修改

    git clone

  • 信息设置

    git config --global user.name 用户名

    git config --global user.email 邮箱

3.markdown

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。可使用HTML的注释、标签。

编辑器：typora是一款所见即所得的MarkDown文本编辑工具，在输入标记符号后，软件会自动对所标记的文本进行渲染，生成相应的格式。

  • 标题

    1、使用 = 和 - 标记一级和二级标题

    2、使用 # 号标记

    ​ 使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。

  • 段落

段落换行：连续回车或使用两个以上空格加上回车

  • 分割线
  • 列表
  • 文字格式
  • 链接

  • 表格

    | 来分隔不同的单元格，使用 - 来分隔表头和其他行。

  • 代码

    代码区块使用 4 个空格或者一个制表符（Tab 键）。

    代码句用反引号（`），就是英文状态下的波浪线。

    代码段用``` 包裹，并指定一种语言。
